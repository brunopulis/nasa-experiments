# Nasa Experiments

Example to Connect Nasa API.

## How to use

To run locally this project clone this repository with:

``` git clone git@github.com:brunopulis/nasa-experiments.git ```

## Installing dependencies 

``` npm install or npm i ```

## Start the app

``` npm run start ```
