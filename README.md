# Nasa Experiments

Example to Connect Nasa API Apod.

## Enviroment

I using this enviroment

* Pingy
* Babel 
* Webpack
* PUG 
* Sass

## Installing Pingy

Run this command

``` npm install -g @pingy/cli ```

## Installing dependencies 

``` npm install or npm i ```

## Start the app

``` npm start ```
