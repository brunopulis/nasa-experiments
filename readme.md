# Nasa Experiments

Simple example using ReactJS with axios

## Enviroment

I using this enviroment

* Babel
* Webpack
* ReactJS
* Axios

## Clone the repository

Run this command

``` git clone git@github.com:brunopulis/nasa-experiments.git ```

## Installing dependencies

``` npm install or npm i ```

## Start the app

``` npm run start ```


Enjoy :smile:
